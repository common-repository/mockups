Mockup Blocks for WordPress Gutenberg featuring 6 free iPhone X mockup photos.
 
== Description ==

Showcase your app and UI right from WordPress.
 
*   Choose from a selection of beautiful and free iPhone X mockup photos
*   Simply add your screenshot, no need to open your image editor
*   Works with themes that support Gutenberg Blocks
 
== Installation ==
 
This plugin requires WordPress Gutenberg. Gutenberg is available as part of WordPress 5.0 and later.
Once you have [Gutenberg](https://wordpress.org/gutenberg/ "Gutenberg") installed, 
 
1. Upload plugin ZIP file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==
 
This plugin works best with a Gutenberg compatible theme. 
 
1. View or create a post or page
2. Add a new device mockup block
3. Upload a screenshot to the block, ensuring that the correct image size is used
4. Choose from one of the mockup images available in the block inspector

== Frequently Asked Questions ==
 
= Does this plugin work without WordPress Gutenberg? =
 
No, it does not. You can, however, install Gutenberg as a separate plugin for WordPress installations lower than 5.0.
See the [Gutenberg](https://wordpress.org/gutenberg/ "Gutenberg") documentation on how to do so.
 
== Screenshots ==
 
1. Choose from a selection of beautiful mockup device images.
2. After installing the plugin, add the mockup block to your post or page.
3. Select your screenshot while making sure it has the specified size and proportions.
4. A preview of your screenshot in the WordPress editor.
5. Select other mockup images using the block inspector.
6. Your post or page with a mockup image showcasing your design.
